Transform your regular link to popup automatically.
 
== Description ==
 
Transform all your regular links to popup automatically using Magnific Popup Javascript library.
 
== Installation ==
 
1. Upload `link-to-popup` directory to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress

== Frequently Asked Questions ==

= What library does this plugin use? =
